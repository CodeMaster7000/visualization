## Visualization

Data analysis tool and tutor to learn chess.

### Install and Setup

Create an `.env` file.

```
cp .env.example .env
```

Install the Composer packages.

```
composer install
```

### Run the App

Run PHP's built-in webserver.

```
php -S localhost:8080 -t public
```
